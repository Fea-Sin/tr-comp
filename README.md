## tr-comp

A template UI component for React using TypeScript

## ✨ 特性

- 📦 打造开箱即用的高质量 React 组件
- ⚙️ 单元测试有利于提高组件可用性
- 🛡 使用 TypeScript 开发，提供完整的类型定义文件
- 🎨 完备 prettier 有利用提高代码可读性与一致性
- 🛠 完备 eslint 代码检查有利于提高代码质量

## 🔨 本地开发

```bash
 git clone https://github.com/Fea-Sin/tr-comp.git your-component-name

 npm install

 npm start
```

## 📦 compile & publish

```bash
npm run compile

npm run pub
```
